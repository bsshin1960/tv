# TV 방영 프로그램 서비스 (TV ON) - 개발 계획 & 도움말 (Plan & Help)

이 문서는 사용자의 요구사항(지상파 하위 서브 풀다운 메뉴 신설, 공중파 삭제, OCN/EBS 방송국 추가, 누누TV 기반 영화/다큐 방송 내용 구성, 체크버그 완벽 수정)에 맞춰 고도화된 **"TV ON"** 웹 애플리케이션의 개발 내역 및 사용 설명서입니다.

---

## 1. 개발 완료 요약 & 빌드 검증

'공중파' 대분류를 없애고 지상파 채널들을 단일화하여 2단계 풀다운 메뉴로 구조화했습니다. 또한, 드롭다운 체크 아이콘이 팽창하여 붉은 박스로 레이아웃을 무너뜨리던 SVG 찌그러짐 현상을 CSS 전용 클래스를 통해 해결했습니다.

### 빌드 결과 확인
* **빌드 명령어**: `npm run build`
* **검증 결과**: 타입스크립트 컴파일 및 Vite 번들링을 무결하게 통과하여 정적 배포 준비 완료.
* **산출물**:
  * PWA 독립 구동 및 오프라인 precaching 용 `dist/sw.js` 및 `dist/manifest.webmanifest`
  * 정돈된 CSS 변수와 2단계 드롭다운 전용 반응형 스타일 시트 (`dist/assets/`)

## 3. 핵심 UI 조작 매뉴얼 (Help)

### A. 카테고리 선택 및 지상파 2차 풀다운 (Sub-Dropdown) Menu
* **설명**: 화면의 산만함을 없애기 위해 '공중파' 항목을 제거했습니다. 이제 대분류는 `전체`, `지상파`, `인터넷 방송`, `홈쇼핑 방송`, `즐겨찾기`로만 간결하게 구성됩니다.
* **사용법**:
  1. 헤더 좌측의 `[카테고리명] 방송 목록 ▼` 버튼을 클릭하여 드롭다운을 엽니다.
  2. 목록에서 **[지상파]**를 선택합니다.
  3. 지상파 선택 즉시 그 우측에 **[KBS 채널 선택 ▼]** (또는 해당 채널명) 이라는 2차 서브 드롭다운 메뉴가 나타납니다.
  4. 이 서브 드롭다운을 열면 **KBS, MBC, SBS, OCN, EBS** 채널이 나열되며, 터치/클릭 시 즉시 해당 실시간 방송이 플레이어에 적재됩니다.
  5. 체크 아이콘(SVG) 크기가 14px로 고정되어, 이전처럼 붉은 활성화 박스로 메뉴를 덮던 시각적 오류가 완벽히 소멸되었습니다.

### B. 신규 추가된 누누TV 벤치마킹 채널 (지상파 소속)
* **🎬 OCN (영화/스릴러/액션)**
  * **설명**: 최신 극장 화제작 및 액션 스릴러 명작 영화들로 편성표를 가득 채운 프리미엄 시네마 채널입니다.
  * **편성표 예시**: *범죄도시 3*, *서울의 봄*, *아바타: 물의 길*, *존 윅 4*, *공조 2: 인터내셔날* 등 극장용 인기 콘텐츠가 시간에 따라 자동 계산되어 방영됩니다.
* **🎓 EBS (교육/다큐/애니메이션)**
  * **설명**: 어린이 교육용 프로그램부터 온 가족이 볼 수 있는 고품격 다큐멘터리까지 포괄하는 에듀 컬처 채널입니다.
  * **편성표 예시**: *자이언트 펭TV (펭수)*, *다큐프라임 (인류의 진화)*, *딩동댕 유치원*, *EBS 스페이스 공감*, *세계 테마 기행* 등이 제공됩니다.

### C. 비디오 플레이어 설정 옵션 풀다운 Menu
* **설명**: 톱니바퀴 모양의 **[설정 옵션]** 버튼을 통해 아래 설정을 일원화하여 제공합니다:
  * **비디오 화질**: `Auto`, `1080p`, `720p`, `480p` 수동 대역폭 전환.
  * **시네마 필터**: 눈 피로 완화(`눈보호`), 다크 시네마(`시네마 흑백`), 차가운 톤(`차가운 블루`).
  * **화면 밝기**: 슬라이더를 통해 플레이어 자체 밝기를 20%~150% 범위로 제어.
  * **취침 타이머**: 예약된 분(`10분`, `30분`, `60분`)이 경과하면 재생이 자동으로 정지.

### D. 에피소드 방영 일정표 아코디언 (EPG)
* **설명**: 플레이어 바로 밑에 위치한 **[오늘의 에피소드 방영 일정표 보기]** 바를 클릭하면, 누누TV 회차 리스트 스타일로 오늘 방영 목록이 펼쳐집니다.
* **알림 예약**: 앞으로 상영할 미래 프로그램 우측의 **종소리(🔔)** 아이콘을 탭해 두면 방송 시간이 되었을 때 시스템 푸시 알림이 작동합니다.

## 4. 휴대폰/PC 기기별 홈 화면 아이콘 설치(PWA) 방법

브라우저의 주소창 및 하단 네비게이션을 숨겨 화면 전체를 Full HD 시네마 비율로 감상할 수 있습니다.

* **안드로이드 크롬(Chrome)**: 주소창 우측의 더보기 메뉴에서 **'앱 설치'** 또는 **'홈 화면에 추가'**를 선택합니다.
* **아이폰 사파리(Safari)**: 하단 툴바의 **'공유'** 아이콘을 누른 후 **'홈 화면에 추가'**를 탭합니다.
* **PC 크롬/엣지(Chrome/Edge)**: 헤더 우측의 **[앱 다운로드]** 버튼 또는 브라우저 주소창 우측의 설치 아이콘을 클릭합니다.
